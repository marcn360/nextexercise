## QuizApp GUI
A GUI for a 12-question short quiz

###
Instructions when running locally
* Install NPM
```bash
npm install
```
* Run locally
```bash
npm run dev
```
### Routes
* /<br>
  Home/Landing
* /Quiz <br>
  Quizzes
### Libraries Used
* tailwind CSS: CSS library for responsive web design
* App Router: for folder based routing
